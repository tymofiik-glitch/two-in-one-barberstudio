# Two in One Barberstudio

Official website for Two in One Barberstudio in Rijswijk.

## Features
- Modern, premium design
- Multi-step booking system
- Bilingual support (NL/EN)
- Responsive layout
- Google Maps integration

## Tech Stack
- HTML5
- TailwindCSS (via CDN)
- Vanilla JavaScript
- Google Fonts (Montserrat & Cormorant)
